Final Project Proposal: Timely

Project Concept/Motivation

I want to build a desk-sized digital clock that shows the current time but also lets the user set a countdown timer and an alarm for specific times (like wake-up or study breaks). When the timer or alarm goes off, the buzzer will sound, or the LEDs will flash to notify.

Anticipated Bill of Materials

  • Arduino Uno (microcontroller)
  • 1x 16x2 LCD display (for time, alarm, and timer display)
  • 4x Push buttons (for mode selection, time set, and start/stop)
  • 1x buzzer (for haptic alarm feedback)
  • 1x RGB LED (for visual alarm feedback)
  • Resistors (for LED, button, LCD display)
  • Breadboard and jumper wires (to connect my wires and the microcontroller)
  • RTC (for real-time display)
  • Laser-cut wood or 3D-printed enclosure

Timeline

  • Set up RTC module and display current time on LCD
    • If not using RTC, use Arduino millis() function as an alternative
  • Build countdown timer function with buzzer and LED
    • If timer logic malfunctions, use a fixed demo duration
    • If buzzer failed, use LED for feedback only
  • Add button controls for mode selections and implement alarm logic
    • If alarm is too complicated, do timer only
  • Assembly the enclosure
    • If not laser-cut or 3D-printed, use simple cardboard
