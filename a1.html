A1: Blink

Schematic

While using red LEDs, I started with 5V output and my desired current for each LED is around 20 mA. To calculate the appropriate resistor value, I used Ohm's Law R = V / I. R = (5 V - 1.8 V) / 20 mA = 160 Ω, meaning that the resistor should be at least greater than 160 Ω, so I picked a 220 Ω resistor as it's the closest resistor I have and still keeps the LED with a decent brightness.

Circuit

I wired 3 red LEDs in parallel, each with its own 220 Ω resistor, all connected back to the same ground.

Each LED lights up sequentially, first on pin 3, then pin 7, and finally pin 10.

Firmware

Arduino Sketch for A1 Blink 

// declaring an array of LED pin numbers
int leds[] = {3, 7, 10};
// storing the total number of LEDs as a variable
int total = 3;

// runs once when the Arduino is powered or reset and initializes the LED pins as outputs
void setup() {
  // looping through each LED pin
  for (int i = 0; i < total; i++) {
    // setting each LED pin as an OUTPUT
    pinMode(leds[i], OUTPUT);
  }
}

// repeatedly cycles through each LED, blinking them in sequence with 1000 ms delays
void loop() {
  // loop through each LED in the array
  for (int i = 0; i < total; i++) {
    // turning the current LED (leds[i]) on by setting pin to HIGH voltage
    digitalWrite(leds[i], HIGH);
    // wait 1000ms while the LED stays on
    delay(1000);
    // turning the current LED (leds[i]) off by setting pin to LOW voltage
    digitalWrite(leds[i], LOW);
    // wait 1000ms while the LED stays off
    delay(1000);
  }
}

  1. Draw a graph where the X axis is time and the Y axis is voltage across the LEDs. Draw a line for each of your 3 LEDs.
    2.

    In my for loop, I programmed the LEDs to light up sequentially in the order of pin 3, pin 7, and pin 10. I used a delay of 1000 ms to keep each LED on for one second, followed by another 1000 ms delay while it stays off. Therefore, each LED alternates between 0 V (off) and 5 V (on) for 1000 ms at a time, blinking one after another. LED 3 first, then LED 7, then LED 10, repeating every 6000 ms.

  2. How many LEDs could you blink independently with your Arduino? How much current would that draw?
    3.

    The Arduino Uno has 14 digital pins, along with 6 analog pins, giving a total of 20 possible connections for LEDs to blink independently. Since the LEDs are wired in parallel, the overall current draw would just be the sum of the current through each LED. Assuming using red LEDs with a forward voltage of 1.8 V and 220 Ω resistors for all LEDs: the current for each LED is (5.0 V - 1.8 V) / 220 Ω ≈ 14.5 mA. With 20 available pins, the total current is 20 * 14.5 mA, which is approximately 290 mA.
